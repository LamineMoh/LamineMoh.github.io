Key Projects

Ontario 511 ETL Pipeline

Designed and implemented an automated real-time data ingestion system using Python and SQL Server. The pipeline processes traffic and road condition data from multiple sources, performs data validation and transformation, and loads processed data into a centralized data warehouse. This solution reduced manual data processing by 90% and improved data freshness from daily to real-time updates.

Fraud Detection Pipeline

Developed a comprehensive risk analysis and anomaly detection system analyzing 56,000+ financial transactions. Leveraged machine learning algorithms (Isolation Forest, Local Outlier Factor) to identify suspicious patterns and potential fraud indicators. The model achieved 94% precision in fraud classification and prevented significant financial losses through early detection capabilities.

AI-Powered Chatbot

Built an intelligent document retrieval and question-answering system using OpenAI's API, LangChain, and Flask. The system leverages vector embeddings to semantically search across large document repositories and generates contextually accurate responses. Implemented with proper error handling, rate limiting, and logging for production-grade reliability.
